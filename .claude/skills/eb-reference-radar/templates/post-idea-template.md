# Template de ideia de post

Use este template para cada ideia gerada pelo radar. Uma ideia por bloco.

---

## [Nome da ideia]

- **Canal recomendado:** Instagram / LinkedIn / ambos
- **Formato:** reels / carrossel / post estático / story / artigo / newsletter / vídeo curto
- **Objetivo do post:** o que essa peça deve provocar em quem trabalha ou quer trabalhar na allu
- **Texto de gancho:** primeira frase que faz a pessoa parar
- **Resumo do conteúdo:** o que o post mostra, em 2 a 4 linhas
- **CTA sugerido:** ação clara e específica
- **Por que faz sentido para allu:** conexão com bastidor real, cultura, vagas ou pessoas
- **Prioridade:** alta / média / baixa
- **Esforço de produção:** baixo / médio / alto
- **Categoria:** vaga / bastidor / cultura / prova social / evento / institucional / educativo / comunidade

---

## Critérios de qualidade da ideia

Antes de registrar a ideia, confirme:

- [ ] É original e adaptada, não cópia de uma referência
- [ ] Mostra algo real da allu, não um clichê de RH
- [ ] O gancho deixa claro por que parar para ler
- [ ] O objetivo está conectado a atração ou marca empregadora
- [ ] A allu aparece em lowercase
- [ ] Não há travessão (—) no texto
