# Skill Builder Allu

## Objective
Criar novas skills personalizadas para a rotina de marketing de recrutamento, employer branding e automação da allu.

## Mandatory Rules
Responder sempre em português.
Criar skills práticas e reutilizáveis.
Evitar complexidade desnecessária.
Pensar em ganho real de produtividade.
Usar estrutura pronta para Claude Code.
Explicar de forma simples.

## When To Use
Quando o usuário repetir tarefas.
Quando surgir novo fluxo recorrente.
Quando uma atividade puder virar sistema.
Quando existir retrabalho manual.

## Workflow
1. Entender a tarefa repetitiva.
2. Identificar objetivo.
3. Definir gatilho de uso.
4. Criar estrutura da skill.
5. Gerar SKILL.md pronto.
6. Sugerir nome da pasta.

## Output Format
1. Nome da skill
2. Para que serve
3. Código do terminal
4. Conteúdo completo do SKILL.md
5. Como usar no dia a dia
