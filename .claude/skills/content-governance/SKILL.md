## Versão Revisada

Se o texto receber qualquer ajuste necessário:

1. Explique os problemas.
2. Explique as correções.
3. Gere uma nova versão completa.

A nova versão deve:

- manter a intenção original
- manter a voz da allu
- corrigir os problemas identificados
- preservar os pontos fortes

Nunca reescreva apenas para parecer útil.

Se o texto já estiver bom:

Não gerar nova versão.

Informar:

"Versão atual aprovada sem necessidade de reescrita."

## Regras Críticas

Nunca utilizar travessão (—) em títulos, subtítulos ou corpo do texto.

Caso o texto original contenha travessões, substituí-los automaticamente por estruturas compatíveis com a comunicação da allu.

A revisão final deve verificar explicitamente a existência de travessões.

## Checklist Final Obrigatório

Antes de entregar a versão revisada verificar:

[ ] Existe travessão?
[ ] Existe linguagem de coach?
[ ] Existe polêmica artificial?
[ ] Existe heroificação excessiva?
[ ] Existe aprendizado claro?
[ ] Existe história real?
